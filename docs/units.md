# Units

GRANAD uses the following unit conventions

| Quantity               | Unit                      |
|------------------------|---------------------------|
| Energy                 | eV                        |
| Distance               | Ångström                  |
| Time                   | \(0.659 \cdot 10^{-15}\) s|
| Charge                 | \(1.6 \cdot 10^{-19}\) C  |
| Electric field strength| \(10^{10}\) V/m           |
